# Learn Go

Learning Go with AI guidance, coming from a JavaScript/TypeScript background.

## Getting Started

```bash
go run .
```

## Quick Reference

```bash
go run .          # Run program
go build          # Compile binary
go test ./...     # Run tests
go fmt ./...      # Format code
go get <pkg>      # Add dependency
```
